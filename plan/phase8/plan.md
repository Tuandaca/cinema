# Phase 8: Optimization & Deployment
# Giai đoạn 8: Triển khai & Tối ưu hóa

## 📝 Mô tả chi tiết (Detailed Description)
Giai đoạn cuối cùng để tinh chỉnh hiệu năng và đưa sản phẩm lên môi trường Production. Đảm bảo ứng dụng chạy nhanh, bảo mật và chuẩn SEO.

## 🔍 Phạm vi thực hiện (Scope)

### 1. Performance Optimization
- Thực hiện API Caching với Redis.
- Tối ưu hóa SEO Metadata cho từng trang Phim.
- Lazy loading hình ảnh và components để đạt điểm Lighthouse > 90.

### 2. DevOps & CI/CD
- Dockerize ứng dụng với Docker Compose.
- Cấu hình GitHub Actions để tự động deploy lên Vercel (Frontend) và Railway/Render (Backend).
- Kiểm tra bảo mật cuối cùng (Security Audit).
