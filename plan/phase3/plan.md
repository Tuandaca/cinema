# Phase 3: Frontend Core UI & Movie Module
# Giai đoạn 3: Giao diện chính & Module Phim

## 📝 Mô tả chi tiết (Detailed Description)
Xây dựng giao diện nghệ thuật (Cinematic) và chuẩn SEO. Trọng tâm là trang chủ và luồng khám phá phim mượt mà bằng Next.js 15 App Router.

## 🔍 Phạm vi thực hiện (Scope)

### 1. Frontend Foundation
- Khởi tạo Next.js 15 (App Router) bên trong Turborepo.
- Tích hợp TailwindCSS và cấu hình Design Tokens theo [09-ui-ux-spec.md](file:///d:/Projects/cinema/docs/09-ui-ux-spec.md).
- Cài đặt thư viện Component `shadcn/ui` và Lucide icons.

### 2. Cinematic Components
- Xây dựng Layout mượt mà (Glassmorphism Header, Footer).
- Tạo Hero Section với video trailer tự động phát (Muted) và hiệu ứng chuyển cảnh.
- Thiết kế Movie Card với hiệu ứng Hover Zoom và thông tin Rating/Genre động.

### 3. Movie Discovery Flow
- Trang danh sách phim (Now Showing / Coming Soon).
- Trang chi tiết phim (Movie Detail) với thông tin Cast, Trailer Popup, và Reviews.
- Tích hợp React Query để quản lý Server State (Caching & Fetching).
